** File space usage utility like du(1) with pretty colors

#+begin_src sh
$ duse -h
Usage of duse:
  -depth int
    	Depth to traverse files (default 1)
  -rsort
    	Sort in descending order
  -s	Summarise the individual files
  -sort
    	Sort in ascending order
#+end_src

#+begin_src sh
$ duse -s -depth 1 ~/Videos
